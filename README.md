# dwm
## [EN] What is dwm?
dwm (dynamic window manager) is a **open source, light and fast** window manager for X.

## [VI] dwm là gì?
dwm (dynamic window manager) là một trình quản lý cửa sổ mã nguồn mở, nhanh và nhẹ cho X.
